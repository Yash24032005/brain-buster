# 🧠 Brain Buster

**Brain Buster** is a modern, interactive quiz platform designed to help students and learners practice school subjects, coding, aptitude, and reasoning. With a clean UI and a focus on speed, it provides an engaging environment for self-assessment.

Preview url

## 🚀 Features

- **17+ Subject Categories**: Includes academic subjects, technical coding challenges, and logical reasoning.
- **Live Timer**: Built-in countdown to help users improve focus and exam-day speed.
- **Dynamic Scoring**: Instant feedback and score calculation upon quiz completion.
- **Admin Panel**: Dedicated area for managing quiz content and oversight.
- **Dark/Light Mode**: Toggle between themes for a comfortable reading experience.
- **Leaderboard**: Local score tracking to keep users motivated.
- **Responsive Design**: Fully optimized for desktop and mobile devices.

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3 (Modern Flexbox/Grid layouts)
- **Scripting**: JavaScript (ES6+)
- **Styling**: Custom CSS with CSS Variables for theme switching.
- **Storage**: Local Storage (for scores and user sessions).
